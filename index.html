Proyectos

Brain Brokers

Juego híbrido que combina tablero físico, cartas, animaciones 3D y realidad aumentada. Me encargué de toda la arquitectura multi-usuario en tiempo real con Socket.io, el desarrollo completo del backend (Node.js, Express.js) y la programación frontend (Three.js, AR.Frame, HTML/CSS/JS). Implementé la sincronización de estado entre múltiples jugadores, sistema de salas, gestión de recursos (dinero y salud mental) y la integración de la ruleta en AR.

Node.js Socket.io Express.js Three.js AR.Frame JavaScript HTML/CSS Web Storage API

Emprendimiento

Sitio web desarrollado para un emprendimiento de tableros en acrílico. Me encargué del diseño y programación de la plataforma, integrando visualizaciones 3D interactivas utilizando la librería Three.js. También modelé los productos en Autodesk Maya para su visualización web.

HTML CSS JavaScript Autodesk Maya

Sombras en la web

Proyecto de narrativa visual donde realicé el diseño, modelado y rigging de personajes en 3D usando Autodesk Maya. Los renders fueron generados con Arnold Renderer y se complementaron con diseño gráfico en Illustrator. El resultado fue un cómic digital con enfoque cinematográfico.

Illustrator Autodesk Maya

Resumidor de noticias

Aplicación construida en Google Colab para procesar y resumir noticias mediante inteligencia artificial. Entrené y evalué modelos sobre un dataset de más de 200,000 artículos utilizando TensorFlow. Además, diseñé una interfaz funcional en Gradio para facilitar su uso.

Python Gradio TensorFlow

Rutas por Colombia

Aplicación web interactiva que visualiza rutas dentro del territorio colombiano aplicando algoritmos de búsqueda heurística como Voraz y A*. Se utilizó la API de OpenStreetMap junto con tecnologías web como HTML, CSS y JavaScript.

HTML CSS JavaScript

Montecarlo Tree Search

Desarrollo de un videojuego de golf en Unity, implementando una inteligencia artificial basada en el algoritmo Monte Carlo Tree Search (MCTS). El sistema evalúa múltiples trayectorias posibles en tiempo real para optimizar cada disparo, ofreciendo una experiencia dinámica y estratégica.

C# Illustrator Unity

Tipsy

Diseño y desarrollo de una plataforma web para estudiantes, con funcionalidades como registro, inicio de sesión y gestión de formularios almacenados en MySQL. La interfaz fue prototipada en Figma y programada con tecnologías web (HTML, CSS, JavaScript).

Figma HTML CSS JavaScript MySQL
